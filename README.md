# 🌤️ Weather App

A simple weather application built with **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.

## Features

* Search weather by city name
* Displays temperature and weather description
* Simple and responsive UI

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## How to Run

1. Download or clone the repository.
2. Open `whether.html` in your browser.
3. Enter a city name and click **Get Weather**.

## API Setup

Replace the API key in the script:

```javascript
const apiKey = "YOUR_API_KEY";
```
